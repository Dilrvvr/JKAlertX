JKAlertX is An AlertView for iOS. Support User Customization Perfectly.  

iOS弹框，包含alert/actionsheet/collectionSheet(类似分享面板)/HUD四种样式。  

自动适配横屏，完美支持自定义。  

支持链式语法，简单优雅！

安装
==============

### CocoaPods

1. 在 Podfile 中添加  `pod 'JKAlertX'`。
2. 执行 `pod install` 或 `pod update`。
3. 导入 `<JKAlertX/JKAlertX.h>`。

### 手动安装

1. 下载 JKAlertX 文件夹内的所有内容。
2. 将 JKAlertX 内的源文件添加(拖放)到你的工程。
4. 链接以下 frameworks:
    * UIKit
6. 导入 `JKAlertX.h`。

系统要求
==============
该项目最低支持 `iOS 8.0` 。
